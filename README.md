# Calculadora Avanzada de Producción de Energía para América Latina

## Descripción
Esta aplicación de escritorio, desarrollada en C# y WPF, permite calcular y visualizar la producción de energía renovable en varios países de América Latina. La calculadora considera cuatro fuentes de energía renovable: solar, eólica, hidroeléctrica y geotérmica.

## Características
- Selección de países de América Latina
- Cálculo de producción de energía diaria, mensual y anual
- Visualización de resultados en gráficos interactivos
- Consideración de factores estacionales en los cálculos

## Requisitos del Sistema
- Windows 7 o superior
- .NET Framework 4.7.2 o superior
- Visual Studio 2019 o superior (para desarrollo)

## Instalación
1. Clone este repositorio:
   ```
   git clone https://github.com/Raulcadiz/AdvancedEnergyProductionCalculator.git
   ```
2. Abra la solución en Visual Studio.
3. Restaure los paquetes NuGet (especialmente LiveCharts.Wpf).
4. Compile y ejecute la aplicación.

## Uso
1. Seleccione un país de la lista desplegable.
2. Elija una fecha para el cálculo.
3. Haga clic en "Calcular" para ver los resultados.
4. Los resultados se mostrarán tanto en texto como en un gráfico interactivo.
